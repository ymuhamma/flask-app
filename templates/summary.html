Popularity

Semantic UI is an alternative to Bootstrap and provides similiar features. Due to the overwhelming popularity of Bootstrap, Semantic UI is left in the shadow of it's older brother. As a results, Semantic UI is not a very popular frontend framework

Strength

  • Ease of Use - Class names are very intuitive
  • Open Source
  • Clean and Consistent Markup

Weakness

  • Few Recent updates - Latest Github Commit in 2018
  • Missing Many Features of Bootstrap
  • Less Browser Compatibilty

Uses

Semantic UI is used to create responsive web designs and layouts (similiar to Bootstrap), while utilizing builtin styled components.

Examples

  • Snapchat
  • Edabit
  • StopLight

Popularity

Flask is very popular in the Python web development community, and actively seen as the first or the second most popular framework. Its only rival being Django.

Strength

  • Lightweight Framework
  • Jinja2 Template Library - Embedded Pythonic code into HTML templates
  • Control over project resources - Developers can utliize any Python library

Weakness

  • Smaller community as compared to Django
  • Complicated to create large scale web applications
  • Security not builtin - Avaliable through importable libraries

Uses

Flask can be used for creating smaller applications, fast. Although, majority of Flask users use it as a backend API for web or mobile applications

Examples

  • Netflix
  • Lyft
  • Reddit
  • Airbnb
