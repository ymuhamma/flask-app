Link Semantic UI Stylesheet

This method is very simple, yet effective for most users. Semantic UI can be "installed" or "loaded" into any html program using the "<link>" command. The exact tag being used on this page is:

<link rel="stylesheet" type="text/css" href="https://cdnjs.cloudflare.com/ajax/libs/semantic-ui/2.4.1/semantic.min.css">

This tag links all UI components and styling of Semantic UI into the HTML file. Additionally, a developer can opt to bring in certain elements using a link tag like this:

<link rel="stylesheet" type="text/css" href="https://cdnjs.cloudflare.com/ajax/libs/semantic-ui/2.4.1/components/button.min.css">

This tag links only the button styling of Semantic UI. The links for each component can be found on CDNJS's website

Installing Flask

Open the command prompt on your local machine, and ensure python is functioning by typing:

C:\Users\USERNAME>python

If there is an error, redownload Python. If the results look similiar to this continue on with the steps:

Python 3.6.3 (v3.6.3:2c5fed8, Oct 3 2017, 18:11:49) [MSC v.1900 64 bit (AMD64)] on win32 Type "help", "copyright", "credits" or "license" for more information. >>>

Using the command line, we are going to utilize the PIP module built into Python to install Flask. Use the following line (this is case sensitive):

pip install flask

Flask has been successfully installed into your local computer. To utilize the Flask module, simply import it into any python file using the following python code

from flask import Flask
